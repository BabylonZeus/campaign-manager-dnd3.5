How to convert WA in md files :
1) Exporter le monde WA : https://www.worldanvil.com/world/8cb142e6-c3ce-4b1e-b33e-55ff69cd8938/edit
2) Le sauvegarder dans D:\data\DnD\World-Forgotten Realms-2e6
3) From repo root (folder containing convert.js and lore/) :
node convert.js ../World-Forgotten\ Realms-2e6/ ./lore/

How the local RAG installation has been setup:

In WSL:
sudo apt-get install zstd
curl -fsSL https://ollama.com/install.sh | sh
ollama pull nomic-embed-text
Test with : ollama list
mkdir -p /mnt/d/data/DnD/campaign/rag-server/pdfs
cd /mnt/d/data/DnD/campaign/rag-server
npm install
cp /mnt/d/data/DnD/campaign/rag-server/setup/index.js /mnt/d/data/DnD/campaign/rag-server/setup/ingest.js /mnt/d/data/DnD/campaign/rag-server/setup/package.json /mnt/d/data/DnD/campaign/rag-server/
cd /mnt/d/data/DnD/campaign/rag-server
npm install
Copy all pdf in rag-server/pdfs

Index pdf files:
cd rag-server
node ingest.js ./pdfs
(Ollama must be running on localhost:11434 for ingest and for Cursor MCP search.)
Cursor MCP: .cursor/mcp.json runs rag-server/index.js (tool search_rules) — edit paths if repo root moves.
